## Features

- Hybrid Detection Engine combining ML predictions with rule-based keyword analysis
- Real-time fake/real job classification
- Risk assessment with Low, Medium, and High Risk output
- Confidence scoring for prediction results
- Premium dark-themed interactive dashboard
- Developer mode for technical details and debugging
- Pre-loaded sample data for testing
